## 공룡 로스터 (1챕터)

### 🌿 초식 — 덮어쓰기: 같은 몸이 강해짐

| ID | 이름 | 코스트 | ATK/HP | 정체성 |
|---|---|---|---|---|
| C001 | 트리케라톱스 | 1 | 4/20 | 밸런스 탱커 |
| C002 | 스테고사우루스 | 1 | 5/17 | 밸런스 |
| C003 | 안킬로사우루스 | 2 | 3/25 | 극탱커 |

**덮어쓰기 규칙**
- 같은 카드를 또 내면 현재 공룡에 **+1 ATK / +3 HP** 영구 누적
- HP 보너스는 현재 HP에도 +3 추가 (몸이 커진 만큼만 얻음, **풀 회복 아님** — 기존 상처는 유지)
- 이름·외형·카드 id 불변
- 스택 상한 없음 (무한 성장)
- 카드는 `discard`로 (재사용 가능)

### 🦖 육식 — 융합: 같은 종 2마리를 합쳐 다음 티어로

| Base | T1 (T0+T0 융합) | T2 (T1+T1 융합) |
|---|---|---|
| C004 랩터 1코 7/13 | **알파 랩터** 10/15 | **절멸 랩터** 13/18 |
| C005 카르노 1코 8/10 | **폭풍 카르노** 11/12 | **심연 카르노** 15/14 |
| C010 콤프소 0코 2/5 | **영리한 콤프소** 3/8 | **무리의 콤프소** 4/12 |

**융합 규칙**
- 육식공룡은 **같은 종 카드를 여러 장 쓴다고 자동 합성되지 않는다** — 필드에 같은 id 육식이 있어도 일반 소환처럼 슬롯 규칙을 따른다.
- 티어 업그레이드는 오직 **융합의 각인 (MAGIC/FUSION)** 카드로만 발동.
  - **C152 융합의 각인** (UNCOMMON): 같은 종·같은 티어 육식 2마리 지정 → 한 단계 위 티어로 진화. **T0+T0→T1**, **T1+T1→T2**. 혼합 티어(T0+T1) 불가.
- 재료는 **필드 / 손패 자유 조합** (필드+필드, 필드+손, 손+손 모두 허용).
- **코스트 규칙: 기본 1코 + 손에 있는 재료의 소환 코스트를 추가 지불.**

  | 재료 위치 | 예시 | 코스트 |
  |---|---|---|
  | 필드 + 필드 | 랩터 T0 둘 | 1코 |
  | 필드 + 손 | 랩터 T0 필드 + 랩터 손 | 2코 |
  | 손 + 손 | 랩터 T0 둘 손 | 3코 |
  | 손 + 손 (콤프소 0코) | 콤프소 T0 둘 손 | 1코 |

- 재료 둘 다 소비: 필드 공룡은 사라지고(해당 카드 → discard), 손 카드도 소진 후 버림더미로.
- 진화 시 **현재 HP 비율 유지 + 최소 maxHp/3 보장**, 누적된 ATK/HP 보너스(버프·영구 강화)는 **둘 중 더 높은 쪽 기준**으로 승계.
- 융합의 각인 자체는 공룡을 소환하지 않는 순수 촉매 — 사용 후 버림더미로 간다.

**필드 공룡 UI 표시**
- 육식: `T0` / `T1` / `T2 · MAX` — 현재 티어만 표시 (스택 개념 없음).
- 초식: `스택 N` — 덮어쓰기 누적 카운트 (스택 > 0일 때).

## 시작 덱 (10장)

**초식 조련사**
- 트리케라톱스 ×2
- 스테고사우루스 ×2
- 작열 송곳니 ×2
- 룬 보주 ×2
- 공격 강화 ×1
- 공룡 치유 ×1

**육식 사냥꾼**
- 랩터 ×2
- 카르노타우루스 ×2
- 작열 송곳니 ×2
- 룬 보주 ×2
- 공격 강화 ×1
- 융합의 각인 ×1

시작 덱에 **융합의 각인 1장 보장** — 첫 런에서 T0+T0→T1 융합 최소 1회 경험. T2(T1+T1 융합)는 보상·상점에서 융합의 각인을 추가 획득 + T1 둘을 구성해야 달성.

## 전투 규칙

| 항목 | 값 |
|---|---|
| 시작 HP | 70 (소환사) |
| 마나 | 매 턴 3 리셋 |
| 드로우 | 매 턴 5 |
| 필드 공룡 동시 수용 | 2체 |
| 덱 구조 | 단일 (deck / hand / discard / bound) |
| 공룡 사망 시 | 소환 카드 → discard 복귀 |
| 필드 꽉 찬 상태 소환 | 기존 공룡 선택해 **교체** (해당 카드 → discard) |
| 같은 종 중복 소환 | 초식=즉시 덮어쓰기(스택 누적). 육식=일반 소환(슬롯 규칙 따름). 육식 진화는 **융합의 각인** 카드로만 (같은 종 2마리 합성). |

### 한 턴 흐름

1. **턴 시작** — 마나·공룡 방어도·공격 플래그 리셋 / 적 인텐트 공개 / 5장 드로우
2. **플레이어 페이즈** — 카드 플레이 (마법 / 버프 / 소환)
3. **공룡 수동 공격** — 공룡 클릭 → 적 클릭으로 명령
4. **턴 종료 자동 공격** — 공격 안 한 공룡이 랜덤 타겟팅 적에 자동 공격
5. **적 페이즈** — 인텐트에 따라 행동 (공격 계열만 lunge 애니메이션)
6. **패 정리** — 손패 → discard, 다음 턴 드로우

### 데미지 계산

- 대상: `block` 먼저 흡수 → 초과분이 `hp`에 차감
- 공룡 방어도는 **매 턴 0 리셋** (장갑 유물 예정)
- 적의 공격 타겟 분배: 도발 공룡 우선, 없으면 공룡 70% / 플레이어 30%

### 적 인텐트 표시

- 공격(⚔) / 방어(🛡) / 강화(▲) / 디버프(▲) / 카운트다운(⚠) 아이콘
- 공격 계열 인텐트: 타겟 힌트 표시 ("→ 공룡 70% / 플레이어 30%" 등)
- 위치: 적 머리 위 + 아이콘 위 (스프라이트에 가려지지 않도록 재조정됨)

## 현재 상태 / 남은 숙제

### ✅ 확정 구현
- 캐릭터 2테크 분기 + 선택 UI
- 덮어쓰기(초식)
- 융합(육식) — C152 융합의 각인 단일 카드로 T0+T0→T1 / T1+T1→T2. 필드/손 자유 조합, 기본 1코 + 손 재료 소환 코스트 추가 지불. 육식 시작 덱은 C202 힐 제거 후 C152 포함.
- 방어 카드 1단계 (C112, C114 + 기존 강화)
- 힐 공룡 전용
- 노드 선택 reachability 수정
- 적 인텐트 타겟 힌트 표시
- 공룡 수동 공격 lunge 애니메이션
- 턴 종료 시 공격 안 한 공룡 자동 랜덤 공격
- 비공격 인텐트(DEFEND/BUFF 등)는 lunge 없이 효과만 적용

### ⚠️ 임시 에셋
- 진화체 카드 아트 (C004_T1~T2, C005_T1~T2, C010_T1~T2) — 텍스트 폴백
- 신규 스펠 아트 (C111, C112, C114) — 텍스트 폴백
- 캐릭터 포트레이트 (CH002는 CH001과 공유 중)

### 🔜 다음 우선순위

1. **방어 2단계** — C113 영구 장갑 (매 턴 +N block) / C115 이번 턴 피해 절반
2. **엘리트 1~2종 추가** — 런마다 변수 확보 (E104, E105)
3. **보스 E901 HP 조정** — 250 → 170 전후 (퍼즐 길이 단축, 기믹은 유지)
4. **일반 적 소소한 기믹 추가** — E002 방어도 유지 격노, E010 회피 2회 후 강타 등
5. **고점 레버 확장** — 테크트리 노드 / 1챕터 유물 3~5개
6. **융합 UX 다듬기** — 융합 카드 코스트 표시가 선택 재료에 따라 동적으로 업데이트되도록 (현재는 base 1코만 표시, 실제 코스트는 확정 시점 계산). FusionSigil.png 아트 에셋 생성.

### 💡 잠재 확장

- 아키타입 시드 카드 (초식 전용 / 육식 전용 버프)
- 이벤트 분기로 특정 공룡 카드 직접 획득
- 유물 효과: "턴 시작 시 모든 공룡 +1 block" 같은 스케일링
- 소환사 포션 풀 확장 (현재 HP 회복 수단 부족)
