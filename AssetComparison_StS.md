# 1챕터 배경 에셋 — StS vs DianoCard

## 슬레이 더 스파이어 Act 1

| 씬 | 배경 |
|---|---|
| 맵 | 1 |
| 일반/엘리트 전투 | 1 베이스 + 전경 프롭(기둥/횃불/돌) 랜덤 배치로 3~5종처럼 보임 |
| 보스 | 3 (보스별 고유) |
| 휴식/상점/보물 | 전 챕터 공용 |

**총 고유: 약 5~6장 (맵1 + 전투1 + 보스3)**

## DianoCard 1챕터 (목표)

| 씬 | 배경 |
|---|---|
| 맵 | 1 |
| 일반 전투 | 3 (통짜 바리에이션) |
| 엘리트 | 1 |
| 보스 | 1 |
| 휴식 | 1 |

**총: 7장**

## 차이

- **StS**: 베이스 + 프롭 조합 (모듈 방식)
- **DianoCard**: 완성된 배경 여러 장 (통짜 방식) — 혼자 개발 + AI 생성에 유리
- 시각 변형 수는 비슷, 제작 방식만 다름
