# 테크트리 기획 — 개념 초안

> 메타 진행(Meta Progression) 축. 여러 Run을 거치며 **영구적으로 캐릭터의 기반 능력치·해금 풀**이 성장하는 시스템. Run 내 덱빌딩과는 별개의 장기 성장 축.

---

## 0. 한줄 요약

**특정 조건 달성 → 속성 포인트 1 획득 → 해당 속성 트리의 노드에 "찍기" → Run 영구 버프/해금 적용.**
스탯형 RPG식(힘/방어/지혜/민첩/의지) 5개 브랜치로 나뉘고, 각 브랜치에는 3~5 티어의 노드가 달림. 포인트는 **브랜치 전용 통화**라 스탯 성장 방향을 플레이어가 선택.

---

## 1. 핵심 설계 원칙

1. **조건 = 실력 증명**. 랜덤 드랍 아님. "이걸 해냈으니 포인트 줌" 식의 명시적 보상.
2. **포인트는 속성 전용 통화**. 힘 포인트는 힘 트리에만 쓸 수 있음. 속성별 조건이 달라 **"어떤 플레이 스타일을 고수하는 플레이어가 해당 속성을 키운다"**는 자연스러운 연결.
3. **성장은 영구**. 찍은 노드는 모든 후속 Run에 적용. 덱빌딩 성장(1 Run 한정)과 완전 분리.
4. **조기 한계 없음**. 모든 노드는 난이도 조건만 맞으면 순서 없이 해금 가능 (브랜치 내 전제 조건 있음).
5. **스탯 버프는 온건**. 한 노드당 +10~20% 이내. 5개 브랜치 풀 해금 시에도 1 Run이 "쉬워지는 것"이 아니라 "선택지가 늘어나는 것"으로 체감.

---

## 2. 5대 속성 브랜치

| 브랜치 | 테마 | 강화 축 | 포인트 획득 성향 |
|--------|------|---------|------------------|
| **힘 (Might)** | 정공법, 폭발 딜 | 공룡 ATK / 공격 마법 피해 / 강력 카드 해금 | 공격적 클리어, 단발 고뎀 달성 |
| **방어 (Fortitude)** | 방벽, 생존 | 최대 HP / 시작 block / 방어 카드 해금 | 저피격 클리어, 도발 유지 플레이 |
| **지혜 (Wisdom)** | 자원, 마법 | 시작 마나 / 드로우 / 유틸 카드 해금 | 많은 카드 사용, 정화/드로우 활용 |
| **민첩 (Agility)** | 속도, 회피 | 턴당 추가 드로우 / 카드 코스트 할인 / 회피 기믹 | 빠른 클리어 (턴수 적음), 약화/회피 활용 |
| **의지 (Will)** | 정신, 상태이상 저항 | 디버프 저항 / 정화 강화 / 특수 유물 해금 | 상태이상 극복 플레이, 독/침묵/취약 버티기 |

> 색상 테마(예시): 힘=적, 방어=청, 지혜=보라, 민첩=녹, 의지=금. 트리 UI에서 분기 구분용.

## 3. 포인트 획득 조건 (예시)

### 3.1 속성별 예시 조건

| 속성 | 조건 예시 (충족 시 해당 속성 포인트 +1) |
|------|-----------------------------------------|
| **힘** | 한 턴에 30 피해 이상 누적, 단일 공격으로 15+ 피해, 보스 처치, 공격 마법만으로 일반 전투 클리어 |
| **방어** | 한 Run에서 피격 0턴 5회, HP 풀로 전투 종료, 도발 카드 N회 성공적 사용 |
| **지혜** | 한 턴에 7장 이상 사용, 정화로 강탈 공룡 회수 성공, 한 Run에서 유틸 카드 20회 사용 |
| **민첩** | 5턴 이내 전투 종료, 한 Run에서 회피/약화 15회 트리거, 카드 코스트 할인으로 대형 콤보 성공 |
| **의지** | 독 상태로 전투 승리, 취약 상태로 전투 승리, 침묵 걸린 채 전투 클리어 |

### 3.2 공통 조건 (브랜치 자유 선택)

- **보스 첫 격파**: 해당 Run에서 가장 많이 사용한 속성 브랜치에 포인트 +1 (자동 태그). 없으면 무작위 1포인트.
- **챕터 완주**: "내가 선택한" 브랜치 1개에 포인트 +1 (확정 보상).

> 수치/세부 조건은 **테스트 중 조정**. 초안은 플레이어가 "뭘 해야 하는지" 직관적이게 설계.

## 4. 노드 티어 구조 (예시 — 힘 브랜치)

각 브랜치는 **3~5 티어**. 상위 티어는 하위 티어의 일부 노드를 전제로 함.

```
힘 브랜치 (예시):
[T1] 공룡 시작 ATK +1
   └─ [T2A] 공격 마법 피해 +1    └─ [T2B] 강력 카드 "돌격" 해금
        └─ [T3] 단일 공격 시 크리티컬 10% (×1.5)
             └─ [T4] "광폭" 카드 해금 (턴당 2번 공격 가능)
                  └─ [T5 ULT] 공룡 최대 공격력 +30% + "원시의 힘" 유물
```

- **T1**: 1 포인트. 작은 영구 버프.
- **T2**: 1~2 포인트. 선택적 분기 (예: 스탯 업그레이드 vs 카드 해금).
- **T3**: 2 포인트. 조합 효과 (크리티컬/연쇄).
- **T4**: 3 포인트. 신규 카드·기믹 해금.
- **T5 궁극**: 5 포인트. 브랜치 정수(상징) — 유물 + 대형 버프. **브랜치당 1개만 해금 가능.**

각 브랜치 만점: 약 12~15 포인트.
5개 브랜치 전부 만점: 약 60~75 포인트 (장기 목표, 수십 Run).

---

## 5. Run과의 상호작용

**Run 시작 시** 적용되는 것:
- 시작 덱에 해금 카드 추가 (카드 해금 노드가 있을 때)
- 시작 HP/마나/block 수치 보정 (스탯 노드)
- 시작 유물 1개 (T5 궁극 노드)

**Run 중** 적용되는 것:
- 드로우 수, 코스트 할인, 크리티컬 확률 등 상시 패시브
- 스탯 기반 보너스 피해·저항

**Run 종료 시**:
- 조건 검사 → 포인트 지급 (레벨업 UI로 알림)
- 로비에서 테크트리 UI로 진입해 찍기

## 6. UI 초안

- **로비 진입 버튼**: "테크트리" (훈련장 버튼 옆)
- **화면 구성**: 중앙 캐릭터 초상 주위로 **5개 방사형 브랜치**. 각 브랜치가 꽃잎처럼 펼쳐짐.
- **노드 상태**:
  - 🟡 **해금 가능** (포인트 충분 + 전제 충족): 노랗게 빛남, 클릭 시 찍기
  - 🟢 **이미 찍음**: 녹색, 효과 툴팁 표시
  - ⚪ **미해금 (전제 부족)**: 회색, 전제 노드 요약 표시
- **포인트 카운터**: 화면 상단 5개(브랜치별). 보유 포인트 명시.
- **조건 목록**: 화면 하단 탭 — 아직 달성 안 한 조건 목록 + 진행도 (예: "5턴 이내 클리어 1/3회")

## 7. MVP 범위 (1챕터 기준)

**지금 만들 것**:
- 3개 브랜치만 구현: **힘 / 방어 / 지혜** (각 T1~T2까지, 노드 2~3개)
- 조건 6~8개 (속성당 2~3개)
- 로비에서 테크트리 화면 진입
- Run 결과에 따라 조건 자동 검사 → 포인트 지급

**2챕터 이후**:
- 민첩/의지 브랜치 추가
- T3~T5 노드 확장
- 궁극 노드 + 유물 연동

## 10. 오픈 이슈

- [ ] 포인트 지급이 **자동 알림**인지 **Run 끝 화면에서 일괄 표시**인지 — UX 결정
- [ ] 노드 "리셋(respec)" 허용 여부 — 있으면 유연성↑, 없으면 선택의 무게↑
- [ ] 포인트가 브랜치 전용이 아니라 **공통 포인트**로 통합하는 대안 — 단순하지만 테마 약해짐
- [ ] T5 궁극 노드 — 브랜치당 1개 제한 엄수인지 조건부 해제인지
- [ ] 캐릭터별 테크트리 분리 여부 (MVP는 고고학자 1명뿐이라 단일 트리)
- [ ] 조건 충족 기록의 **Run 간 영속성** — 조건은 Run 단위인지 영구 누적인지 (예: "도발 5회"가 한 Run 안인지 누적인지)
