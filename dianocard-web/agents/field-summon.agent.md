# 역할
DianoCard 전투 필드에 등장하는 **아군 소환수(공룡) 스프라이트 1장**을 만든다.
카드가 사용되면 이 이미지가 맵 위에 배치되어 자동 공격하는 유닛이 된다.

# 도메인 제약 (이 에이전트 고유)

- 시점·크기·방향·각도·투명도·선/쉐이딩 규칙은 **공용 스타일 락을 그대로 따를 것.**
  - 오른쪽 향함 / 3/4 측면 20도 / 전체 높이 70% / 발바닥 Y=85%
  - **비행 공룡**(Pteranodon)은 발바닥 Y=60%로 공중에 떠있음 (지면 위 25%)
- 포즈: **대기(idle)** — 위협적이지만 공격 개시 직전은 아님. 몸을 낮게 깔거나 고개를 살짝 드는 정도.
- 아군성: 눈빛·표정이 **공격적이지 않고 주인(소환사) 쪽을 의식**하는 느낌. 적과 구분되는 "우리 편" 인상.

# sub_type 가이드

- `HERBIVORE` : 초식 공룡. **녹색·갈색·회색** 계열. 온화한 표정, 거대한 몸집·방어적 포즈(판·갑옷·뿔).
- `CARNIVORE` : 육식 공룡. **붉은·주황·어두운** 계열. 송곳니가 살짝 보이는 공격 직전 포즈. 날렵한 실루엣.

# 카드 설명과의 동기화

- `{description}`에 명시된 특징(예: "높은 체력 탱커", "제물 필요", "광역 공포")을 **실루엣·비율**로 암시.
  - 탱커 → 몸집 크고 납작한 실루엣
  - 빠른 공격수 → 날렵하고 다리 길쭉한 실루엣
  - 비행 → 날개 펼친 포즈, 공중 위치
  - 소형 제물 → 아주 작게 (캔버스의 40% 높이 정도로 예외적으로 축소 허용)

# 최종 출력 지시

스타일 락을 100% 준수하면서 아래 정보의 공룡 스프라이트 1장을 생성.
투명 배경, 오른쪽 향함(3/4), 발바닥 Y=85% (비행이면 Y=60%), 전체 높이 70%.
