# 역할
DianoCard 전투 필드에 등장하는 **적 몬스터 스프라이트 1장**을 만든다.

# 도메인 제약 (이 에이전트 고유)

- 시점·크기·방향·각도·투명도·선/쉐이딩 규칙은 **공용 스타일 락을 그대로 따를 것.**
  - **왼쪽 향함** (아군 공룡과 거울상) / 3/4 측면 20도 / 전체 높이 70% / 발바닥 Y=85%
- 포즈: 대기(idle) 자세. 위협적이지만 공격 개시 직전은 아님.

# enemy_type 치수 조정

**기본 원칙**: 스타일 락의 "세로 70% / 가로 80% 이내 + 좌우 10% 여백" 은 절대 위반 불가.
티어별 크기는 그 범위 안에서만 조정한다.

- `NORMAL` : 세로 **65~70%**, 가로 70~80% 범위. 평범한 위압감.
- `ELITE`  : 세로 **70~75%**, 가로 75~80% 범위. 디테일 많게(뿔·독포자 등). 크기보다 **밀도**로 강함을 표현.
- `BOSS`   : 세로 **75~80%**, 가로 **75~80% 범위 필수**. 카리스마는 실루엣·디테일로 표현하되 **절대 프레임을 꽉 채우지 말 것**. 좌우 여백은 신성함·위엄을 더 돋움.

**와이드 몬스터 주의** (길쭉한 실루엣):
- 뱀형·박쥐형·덩굴형 등 가로가 긴 몬스터는 **세로 60% 이하로 낮추고 가로를 80% 이내**로 맞춤
- 꼬리·덩굴·날개 끝이 프레임 가장자리에 닿으면 위험 → 안쪽으로 말아넣거나 살짝 접힌 포즈로

# 챕터 분위기 톤 — 환경 레퍼런스 기반

환경 레퍼런스 이미지에 첨부된 **실제 배경 맵**이 최종 톤의 진실이다. 아래 챕터 설명은 그 맵들에서 읽어낸 요약일 뿐, 맵과 충돌하면 **맵을 따를 것**.

## 1챕터 — 무너진 정글 유적 (현재 레퍼런스 맵이 커버하는 범위)
- **팔레트**: 이끼 덮인 회갈색 석재(#6b6a55 ~ #3e4034) / 깊은 청록 안개(#3b5c56 ~ #1f3a3a) / 따뜻한 황갈색 바닥(#8b7549 ~ #5a4423) / 드문 붉은 악센트(제단 깃발 #8c2b24)
- **채도·명도**: 중간 채도, **상단은 어둡고 차가움 · 하단은 따뜻하고 밝음**의 수직 대비. 몬스터는 이 대비에 맞춰 상반신 섀도는 청록, 하반신 라이트는 황갈색 계열로 떨어지게.
- **환경 디테일 DNA** (몬스터 표면에 녹아들면 자연스러움):
  - 이끼·수분 낀 돌 텍스처 / 늘어진 덩굴·뿌리 / 양치류 잎 실루엣 / 부서진 돌조각
  - 뒤쪽 깊은 정글의 **청록 안개**가 몬스터 뒤쪽 윤곽선을 살짝 머금게
- **금지**: 레퍼런스 맵의 **구조물 자체**(돌기둥·아치·부조·공룡 갈비뼈·검·깃발)를 몬스터 몸에 이식하지 말 것. 이끼·덩굴·뿌리 같은 **공통 재료**만 빌려옴.

## 2챕터 — 사막·피라미드. **모래·황금·청금석** 톤.
## 3챕터 — 화산·용암. **암적·탄흑·유황황** 톤.
## 4챕터 — 심연·원시. **보라·청흑·원시녹·인광** 톤. (크툴루 미학 최대 농도)

# 🐙 크툴루 미학 — **배경 방사능 수준** 아주 은은하게

> ⚠️ 이 섹션의 전제: **몬스터 90% 이상은 평범한 판타지 생물로 읽혀야 한다.** 크툴루는 **옷**이 아니라 **공기 중 극미량의 입자** — 관객이 의식적으로 "아, 크툴루네"라고 알아채면 이미 실패. "뭔가 미묘하게 어긋난 느낌이 든다" 정도가 성공.

**원형 동물·공룡·파충류의 정체성이 항상 지배적**이다. 크툴루 DNA는 그 위에 얹은 **한 방울의 이물감**일 뿐.

## 허용되는 힌트 (아주 절제된 선택지)
아래 중 **최대 1가지만**, 그것도 **눈에 띄지 않게** 섞을 것:
- **미세한 비대칭**: 좌우 눈 크기가 살짝 다름 / 귀 한 쪽만 약간 변형 (과장 금지)
- **눈동자의 이상함**: 동공 모양이 약간 특이함 (세로 슬릿, 작은 다중 동공) — **눈 개수는 정상**
- **피부의 미묘한 축축함·인광**: 관절 틈새·목 아래 옅게 빛나는 반점 몇 개 (전신 발광 금지)
- **응시의 고요함**: 광기·분노 표정이 아니라 **무표정한 침착함** — 표정 자체가 힌트
- **비늘/껍질의 동심원 패턴**: 평범한 비늘 사이 한 군데만 이상한 나선·점박이

## 절대 금지 (❌)
- **촉수** — 기본적으로 **안 쓴다**. 꼬리 끝·입 주변 어디에도 촉수 덩굴 금지. (BOSS의 특별한 경우에만, 그것도 1~2가닥 짧게)
- **문어 얼굴 / 문어 머리** — 완전 금지
- **추가 팔·다리·머리** — 해부학은 원형 동물 그대로
- **전신 인광 / 반투명 막으로 뒤덮인 몸** — 공포물 레벨 이질감 금지
- 크툴루·다곤·쇼고스·딥원 등 신화 캐릭터의 특징적 실루엣 복제
- 오각별·네크로노미콘 심볼·설형문자 같은 인식 가능한 신화 기호
- B급 펄프 공포 네온색 / 블리딩 레드

## enemy_type 별 농도
- `NORMAL`: **힌트 0개가 기본**. 완전히 평범한 판타지 동물/공룡으로 그릴 것. 굳이 넣는다면 "표정의 고요함" 정도만 — 외형 개조 없음.
- `ELITE`: **힌트 최대 1개**, 그것도 **부분 디테일**로만. 한눈에 보면 그냥 위협적인 동물. 자세히 보면 "어? 눈동자가 좀..." 정도.
- `BOSS`: **힌트 1~2개**까지 허용. 그래도 실루엣의 80%는 원형 생물. "보스라서 좀 더 이상하다"는 인상이 들지만 여전히 괴수물 캐릭터.

## 챕터별 농도 승수
- 1챕터 ×0.3 (거의 안 보임 — 힌트 넣어도 의식 못할 정도)
- 2챕터 ×0.5
- 3챕터 ×0.7
- 4챕터 ×1.0 (심연 챕터만 본격적으로 이질감 허용)

**판단 기준**: 플레이어가 이 몬스터를 보고 "무서운 동물"이라고 부르면 성공. "저거 크툴루 몹이네"라고 부르면 실패.

# 패시브·패턴과의 동기화

- `{passive_ids}`(파이프 구분)에 따라 **실루엣에 힌트 반영**:
  - 이끼 관련 → 몸에 이끼·덩굴 부착
  - 격노·분노 → 핏줄·눈빛 강조
  - 방어 중심 → 갑피·껍질 강조
  - 성장형 → 포자·돌기 같은 "자라나는" 요소
- `{description}`의 키워드가 최우선 — 동물 원형을 먼저 잡고 패시브·크툴루 DNA는 표면 장식으로 얹기.

# 최종 출력 지시

스타일 락을 100% 준수하면서 아래 정보의 몬스터 스프라이트 1장을 생성.
투명 배경, 왼쪽 향함(3/4), 발바닥 Y=85%, enemy_type에 맞는 크기.

**첨부된 환경 레퍼런스(맵)를 실제로 보고** 이 캐릭터가 맵 오른쪽에 얹혔을 때
색·명도·안개 톤이 자연스럽게 녹아들도록 팔레트를 맞출 것 — 단 맵 속 오브젝트는 복사 금지.

크툴루 미학은 **거의 보이지 않을 정도로만** — 대부분의 몬스터는 평범한 판타지 동물이다.
촉수·문어얼굴·추가 사지 금지. 기본은 "위협적인 동물 그대로" 그리고 거기서 멈추기.
