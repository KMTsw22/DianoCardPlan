# 역할
DianoCard 전투 필드에 등장하는 **적 몬스터 스프라이트 1장**을 만든다.

# 도메인 제약 (이 에이전트 고유)

- 시점·크기·방향·각도·투명도·선/쉐이딩 규칙은 **공용 스타일 락을 그대로 따를 것.**
  - **왼쪽 향함** (아군 공룡과 거울상) / 3/4 측면 20도 / 전체 높이 70% / 발바닥 Y=85%
- 포즈: 대기(idle) 자세. 위협적이지만 공격 개시 직전은 아님.

# enemy_type 치수 조정

**기본 원칙**: 스타일 락의 "세로 70% / 가로 80% 이내 + 좌우 10% 여백" 은 절대 위반 불가.
티어별 크기는 그 범위 안에서만 조정한다.

- `NORMAL` : 세로 **65~70%**, 가로 70~80% 범위. 평범한 위압감.
- `ELITE`  : 세로 **70~75%**, 가로 75~80% 범위. 디테일 많게(뿔·독포자 등). 크기보다 **밀도**로 강함을 표현.
- `BOSS`   : 세로 **75~80%**, 가로 **75~80% 범위 필수**. 카리스마는 실루엣·디테일로 표현하되 **절대 프레임을 꽉 채우지 말 것**. 좌우 여백은 신성함·위엄을 더 돋움.

**와이드 몬스터 주의** (길쭉한 실루엣):
- 뱀형·박쥐형·덩굴형 등 가로가 긴 몬스터는 **세로 60% 이하로 낮추고 가로를 80% 이내**로 맞춤
- 꼬리·덩굴·날개 끝이 프레임 가장자리에 닿으면 위험 → 안쪽으로 말아넣거나 살짝 접힌 포즈로

# 챕터 분위기 톤

- 1챕터 : 이끼낀 고대 유적. **녹색·회색** 톤.
- 2챕터 : 사막·피라미드. **모래·황금** 톤.
- 3챕터 : 화산. **붉은·검은** 톤.
- 4챕터 : 심연·원시. **보라·청흑·원시녹** 톤.

# 패시브·패턴과의 동기화

- `{passive_ids}`(파이프 구분)에 따라 **실루엣에 힌트 반영**:
  - 이끼 관련 → 몸에 이끼·덩굴 부착
  - 격노·분노 → 핏줄·눈빛 강조
  - 방어 중심 → 갑피·껍질 강조
  - 성장형 → 포자·돌기 같은 "자라나는" 요소
- `{description}`의 키워드가 최우선 — 동물 원형을 먼저 잡고 패시브는 표면 장식으로 얹기.

# 최종 출력 지시

스타일 락을 100% 준수하면서 아래 정보의 몬스터 스프라이트 1장을 생성.
투명 배경, 왼쪽 향함(3/4), 발바닥 Y=85%, enemy_type에 맞는 크기.

## 대상 정보
- ID: {id}
- 이름: {name_kr} ({name_en})
- 종류: {enemy_type} · 챕터 {chapter}
- 패턴셋: {pattern_set_id}{phase_set_id}
- 패시브: {passive_ids}
- 설명: {description}
