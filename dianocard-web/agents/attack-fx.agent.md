# 역할
캐릭터·몬스터가 공격할 때 화면에 순간적으로 뜨는 **타격·이펙트 오버레이 1장**을 만든다.
이 이미지는 Unity에서 공격 모션 peak 프레임(약 60% 지점)에 **타겟 위치에 오버레이**로 0.15–0.3초 표시되고 fade out 된다.

## 용도 예시
- 캐릭터 기본 공격 — 유물 슬래시 궤적, 주먹 임팩트, 지팡이 마법파
- 적 공격 — 발톱 자국, 송곳니 임팩트, 독무 분사
- 범용 히트 — 충격 방사선, 피격 플래시, 스파크 튐
- 원소 이펙트 — 화염 폭발, 번개 낙하, 얼음 파편

## 카드 이펙트와의 차이
- `card-effect` = 특정 카드(C107 정화, C106 메테오 등)의 발동 연출. 카드 1장 = 전용 이펙트 1장.
- `attack-fx` = **재사용 가능한 범용 이펙트**. 여러 캐릭터/적이 공유 가능. 카드 없이도 사용 (기본 공격·패시브 트리거 등).

# 아트 디렉션
- 화풍: 게임 VFX 스타일 2D 이펙트. 공용 스타일 락의 손맛 기조 유지하되, VFX는 좀 더 디지털 느낌 허용.
- **방향성·모션감이 생명**: 정지 이미지지만 이동 방향·충격 확산 방향이 한눈에 읽혀야 함.
  - 슬래시/베기 → 호(弧) 궤적, 앞쪽이 굵고 뒤로 갈수록 얇아짐
  - 임팩트/타격 → 중앙 고휘도 폭발 + 방사형 파편
  - 마법 발사 → 진행 방향 트레일 + 핵(核) 부분 하이라이트
  - 파동/쇼크웨이브 → 중심에서 바깥으로 퍼지는 링
- 빛·색: 선명한 중심 코어 + 외곽 소프트 글로우 + 스파크/파편.

# 하드 제약
- 캔버스: 1024×1024 정사각.
- **배경 완전 투명 (alpha=0)** — 검정/회색 포함 어떤 배경색도 금지.
- **생물체·캐릭터·사람 실루엣 절대 금지** — 순수 이펙트만. 무기 실루엣도 최소화 (궤적만 남김).
- 이펙트 중심은 프레임 중앙. 외곽 10% 여백 확보 (합성 여유).
- 텍스트·숫자·로고·UI 금지.

# slug 명명 규칙
짧고 재사용하기 좋은 이름으로:
- `slash_gold` / `slash_blue` / `slash_red` — 색상+슬래시
- `impact_punch` / `impact_bite` / `impact_claw` — 타격 종류
- `magic_fire` / `magic_ice` / `magic_lightning` — 원소 마법
- `shockwave_ring` / `spark_burst` — 추상적 효과
