# 역할
몬스터 **패시브**를 상징하는 **작은 원형 뱃지 아이콘 1장**을 만든다.
전투 UI에서 몹 HP 바 바로 아래, 가로로 나열되는 22px 높이 칩 안에 들어가는 뱃지.

# 아트 디렉션
- 화풍: 플랫 일러스트 + 강한 실루엣. Slay the Spire 상태이상 아이콘 톤.
- 색상: 패시브 종류에 따라 상징색 한 가지가 강하게 드러나게.
  - 공격 계열(분노·격노·성장·거대화): 붉은/주황
  - 보호·수호 계열: 녹색/에메랄드
  - 덩굴·자연 계열: 진한 녹색 + 갈색
  - 침묵·저주 계열: 보라/회색
  - 이중 행동·속도 계열: 밝은 노랑/흰색
- 외곽: 어두운 라인 또는 드롭섀도우로 작은 크기에서도 실루엣이 분명.
- 중앙 심볼 1개. 장식 최소화.

# 하드 제약
- 캔버스: **128 x 128 정사각**.
- **배경은 완전 투명**.
- 심볼은 프레임 중앙 70% 영역 안.
- 텍스트/숫자/로고 금지. (Unity에서 덧씌움)
- 좌우 대칭이면 이상적이지만 필수는 아님.

# 참조 예시 (해석 가이드)
| 패시브 예 | 심볼 컨셉 |
|-----------|-----------|
| 쌍둥이의 분노 | 교차된 두 검 + 핏방울, 붉은 톤 |
| 뿌리 번식 | 흙에서 솟는 새싹 두 개, 녹색·갈색 |
| 성장 | 상승 화살표 + 포자, 노랑·녹색 |
| 이끼 수호 | 이끼 낀 방패, 녹색 에메랄드 |
| 이끼 소환술 | 이끼 덩이 + 별/반짝임 |
| 거대화 | 바깥으로 뻗는 네 화살표, 붉은/황금 |
| 이중 행동 | 두 개 원형 화살표 | 
| 공룡 강탈 | 움켜쥔 손 + 덩굴, 보라 |
| 울부짖음 | 확산 파동선, 회색·보라 |
| 취약 각인 | 붉은 균열 X, 붉은 |
| 분할 타격 | 세 개 슬래시, 주황 |
| 재생 장갑 | 방패 + 톱니바퀴, 청회색 |

- ID: {id}
- 이름: {name_kr} ({name_en})
